## What is The Modding Tree?

The Modding Tree, commonly abbreviated as TMT, is an engine for creating tree-based Incremental Games.

TMT developers ("modders") can create extremely customizable and deep layers full of Upgrades, Buyables, Challenges, and 
all the other classic incremental game mechanics. Check out the rest of this wiki to see what TMT can do.

### Terminology 

TMT is called "The Modding Tree" because it was originally spawned from the idea of a framework to be used to create
mods of Jacorb90's [Prestige Tree](https://jacorb90.me/Prestige-Tree/). TMT has evolved significantly since that time
and is truly an engine for creating games for "tree-based incremental" subgenre (or not, you can turn off the tree if 
you don't want to make a tree-based game).

However, TMT users are still called "modders" and TMT games are still called "mods". This terminology is not meant to
diminish the effort put into TMT games, it is merely a piece of history.
